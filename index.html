HTML

  • Hypertext Markup Language (HTML) is used to create the structure of a webpage.
  • HTML elements work together to make a website structed and easy to navigate.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Child elements are held inside parent elements.
  • The section element defines a section in a document that contains thematic content.
  • Semantic HTML elements indicate the roles of their content.

CSS

  • Cascading Style Sheets (CSS) is used to style a webpage.
  • CSS defines the colors, aesthetics, vistual layout, and organization of a webpage.
  • The three ways to style a webpage using CSS are incline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS is used to style an element directly in an HTML file.
  • Internal style sheets contain CSS rules for a webpage in the head section of an HTML file.
  • External CSS style sheets allow developers to keep all their CSS rules in a separate file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -a
  • git add -A: stages our changes
  • git pull origin main: pulls in the base branch to our feature branch
  • git push origin: pushes up your changes to the remote respository
  • git commmit -m "your message here": describes the changes you made

JavaScript

  • Javascript (JS) is used to handle logic and allow users to interact with a website.
  • JS powers your websites with effects, interactivity, and data communication.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements and loops can be used to control the order in which code is executed.
  • Array is a single variable that is used to hold a group of data.
